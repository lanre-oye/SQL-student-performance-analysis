# 🏫 Netfula School Student Performance Analysis
This project demonstrates end-to-end data cleaning and transformation using **MySQL**.

## 🛠️ Key SQL Techniques Used
- **Data Cleaning:** Handled null values and standardized race/ethnicity categories.
- **Feature Engineering:** Created `Total_Score`, `AVG_Score`, and a conditional `Grade` system (A-F).
- **Advanced Joins:** Implemented `LEFT JOIN` to map parental education levels to student performance.
- **Transactions:** Used `START TRANSACTION` and `COMMIT` for safe data updates.

## 📁 Files in this Repository
- `analysis.sql`: The full MySQL script containing the cleaning and analysis logic.
